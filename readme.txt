----------------------------------------------------------------------------------------

To run this project, you need to install the following requirements:
+ Node version 18 or higher
+ "npm install -g nodemon" (Run this command if using Mac)
+ "npm install --save-dev nodemon" (Run this command if using Mac)

----------------------------------------------------------------------------------------

How to run the project:
1. In the source directory:
   Navigate to the Source folder (frontend). To run it, execute the following commands:
   - Run "npm i" or "npm install" to install the dependencies.
   - After installation, run "npm run dev" to start the project.

----------------------------------------------------------------------------------------

2. Additional directories:
   To run the Source folder, you need to start the 5 backend folders.

----------------------------------------------------------------------------------------

3. For each backend folder:
   - Run "npm i" or "npm install" to install the dependencies.
   - After installation, run "npm run dev" to start the server.
   - Repeat the same steps for the remaining backend services.

----------------------------------------------------------------------------------------
